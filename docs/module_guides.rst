Module guides
=============

.. important::
   The documentation from here on out is a great reference tool. However, it is currently undergoing an update so if something you are referring to here does not work - do not worry!

This section contains guides fully detailing the usage of the many python modules within this package. This does not include detailed, implementable workflows and instead focusses on detailed explanations of each method and function available in these modules.

**Module References:**
The modules have been renamed for clarity. For detailed information, see:
- `modules.molecule_builder` - Molecular structure utilities
- `modules.filepath_manager` - File and directory management
- `modules.system_builder` - MD system construction
- `modules.simulation_engine` - OpenMM simulation wrapper
- `modules.trajectory_analyzer` - Trajectory analysis tools
- `modules.charge_calculator` - Multi-method charge calculations
- `modules.quantum_calculator` - Quantum chemistry integration

For backward compatibility, these are still available via old names in the `compat` module.
