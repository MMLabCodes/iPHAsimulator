# Development Notebooks

This folder contains development and experimental notebooks used during the refactoring of the SatisPHAction Simulator project.

## Contents

- **sw_analysis_dev.ipynb** - Development notebook for trajectory analysis module
- **sw_analysis_dev_3.ipynb** - Alternative development version of trajectory analysis

## Purpose

These notebooks were used during development and testing of the `trajectory_analyzer` module refactoring. They are kept for reference but are not part of the main tutorial curriculum.

## For New Users

Please refer to the main tutorials in the `/tutorials/` folder:
- **Tutorial_10_Analysis_module.ipynb** - Official analysis tutorial
- **Tutorial_11_Analysis_module_2.ipynb** - Advanced analysis topics

## For Developers

These notebooks can be useful for:
- Understanding the evolution of the trajectory analysis module
- Testing new features before integration
- Debugging analysis workflows
